Giới Thiệu

Lập trình frontend là một trong những lĩnh vực phổ biến nhất trong ngành công nghệ hiện nay.

Để xây dựng giao diện web, chúng ta thường bắt đầu với HTML, CSS và JavaScript.

Ban đầu việc học có thể khá khó khăn nhưng nếu kiên trì luyện tập mỗi ngày, bạn sẽ dần hiểu cách hoạt động của website.

Một trong những điều quan trọng nhất là phải thực hành thường xuyên.

Chỉ học lý thuyết mà không làm dự án thực tế sẽ khiến bạn khó tiến bộ.

Hành Trình Học Lập Trình

Khi mới bắt đầu, tôi thường sử dụng console.log() để kiểm tra dữ liệu trong JavaScript.

Ngoài ra, việc ghi nhớ các phím tắt như Ctrl + S giúp tiết kiệm rất nhiều thời gian khi code.

Sau khi chạy chương trình, kết quả hiển thị có thể như sau:

Hello World

"Programs must be written for people to read, and only incidentally for machines to execute." - Harold Abelson

Điều tôi học được là việc viết code sạch và dễ đọc quan trọng không kém việc code chạy đúng.

Code Examples

Ví dụ HTML

<!-- Tạo nút button -->
<button>
  Click Me
</button>

Ví dụ JavaScript

// Hàm hiển thị thông báo
function hello(){

  // In ra console
  console.log("Hello World");

}

// Gọi hàm
hello();

Ví dụ CSS

/* Style cho button */
button{
  background: blue;
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
}
